미해결 질의 적재 + PII 스크러빙

전화·주민·카드번호 정규식 마스킹 후 비동기 fire-and-forget INSERT → hot path 지연 없음

backfill_embeddings.py — embedding IS NULL 행을 batch 처리. Gemini Embed 768차원 생성 후 HNSW 인덱스 업데이트

⑤

주간 클러스터링 분석 → 정책 DB 확충
