# B009. 국공립 공연장 입장료 50% 할인

> 전단지 원문: **"국·공립 공연장 50% 할인(심한 장애인은 동행하는 보호자 1인 포함). * 대관공연 제외"**
> 구분: 심한장애(중증) / 심하지 않은 장애(경증) — 전국공통, ⑨ 번 항목
> 최종 확인: 2026-05-19 (v1.0.0)

## 3. 대상자 자격

- **등록 장애인** (「장애인복지법」 제32조).
- **중증(심한 장애)**: 본인 + 동반 보호자 1인까지 같은 50% 할인.
- **경증(심하지 않은 장애)**: 본인만 50%.
- **외국인 등록 장애인**: 「장애인복지법」 제32조의2에 따른 등록자. 외국 발급 장애인증 불가.

---

## 4. 받는 방식 (How to use)

### (A) 온라인 예매 (가장 일반적)
1. 공연장 홈페이지 또는 인터파크·예스24·티켓링크 등에서 공연 선택.
2. 좌석 등급·좌석 선택.
3. **'장애인 우대(또는 장애인 할인)' 옵션 체크** → 50% 할인 결제.
4. 중증은 본인 + 동반자 2장 모두 50% 결제(인접 좌석).
5. 공연 당일 현장 매표소에서 **본인 명의 신분증 + 장애인등록증** 제시 후 티켓 수령.

### (B) 전화·현장 예매
- 콜센터 또는 매표소에 '장애인 우대'로 예매 요청. 등록증 제시 후 50% 결제.

### 대관공연 식별 방법
예매 페이지의 **'주관/주최'** 항목 확인:
- 공연장 이름(예: "예술의전당") → **자체기획** → 50% 할인 적용
- 외부 회사 이름(예: "쇼노트", "마스트엔터테인먼트") → **대관** → 주최측 자체 규정(통상 5~30% 또는 미적용)

### 예외·주의
- **대관공연**은 본 항목 적용 외.
- **휠체어석**은 본 50% 할인과 별개. 좌석 자체가 별도 운영(보통 일반석보다 저렴).
- **인기 공연 VIP·R석**은 우대 할인 제한 좌석 있을 수 있음.
- **본인 명의 결제** 필요. 보호자 명의 예매 시 매표소 거부 가능.

## 8. 자주 묻는 질문 (FAQ)

**Q1. 어느 공연장이 50% 할인 대상인가요?**
A. 국·공립 공연장 — 국립극장(장충동), 국립국악원(서초·부산·진도·남원), 국립극단(명동), 예술의전당(우면동), 세종문화회관(광화문), 정동극장(정동·경주), 그리고 전국 220여 개 공립 문예회관(부산문화회관·대구문화예술회관·광주예술의전당·대전예술의전당 등). 사립·민간 공연장(블루스퀘어·LG아트센터·민간 콘서트홀)은 대상 아님.

**Q2. '대관공연 제외'가 무슨 뜻인가요?**
A. 공연장이 직접 기획·제작하거나 전속·상주 단체가 출연하는 공연(**자체기획공연**)만 50% 할인 대상. 외부 단체(공연 제작사·기획사·민간 프로모터)가 공연장을 빌려 운영하는 공연(**대관공연**)은 주최측 자체 규정(통상 5~30% 또는 미적용). 같은 공연장이라도 둘이 다름. 예매 페이지 '주관/주최' 항목으로 구분 가능.

**Q3. 예매 시 어떻게 50% 할인?**
A. 좌석 선택 후 **'장애인 우대(또는 장애인 할인)' 옵션 체크** → 즉시 50% 결제. 공연 당일 매표소에서 본인 신분증과 장애인등록증 제시 후 티켓 수령. **본인 명의 결제** 필수.

**Q4. 보호자도 50%?**
A. **중증** 동반 보호자 1인까지 같은 50%. 본인 + 동반자 2장으로 좌석 선택 후 우대 옵션 → 둘 다 50%. 보호자는 친인척·간병인·자녀 등 누구든. **경증은 본인만**.

**Q5. 현장에서 복지카드를 안 가져온 게 발견되면?**
A. 티켓 수령 거부 또는 차액(50%) 추가 결제 후 입장. 모바일 복지카드(정부24·복지로 앱) 인정 — 미리 깔아두기. 주민센터 즉시 발급 '장애인 증명서'로 대체 가능(시설별 상이).

**Q6. 외국인도?**
A. 「장애인복지법」 제32조의2에 따라 외국인 등록 후 장애인 등록을 마친 경우 동일 적용. **외국 발급 장애인증은 불가** — 대한민국 등록증 필수.

**Q7. 휠체어석은 어떻게?**
A. **휠체어석은 본 50% 할인과 별개**. 좌석이 별도로 운영되며 일반석보다 저렴한 경우 많음. 동반자 1인까지 인접 좌석 함께 예매 가능. 좌석 수 제한적이므로 공연장 홈페이지 또는 전화 예매 권장(위탁사 예매 불가한 경우 있음).

**Q8. VIP·R석도 50%?**
A. 원칙적으로 모든 등급에 적용되나, 인기 공연은 VIP·R석 일부를 우대 대상에서 제외하는 경우 있음. 예매 페이지에서 우대 가능 좌석 확인.

**Q9. 어린이극·청소년극도?**
A. 정상가가 이미 낮아 50% 대신 정액 할인(예: 2,000원)이거나 미적용일 수 있음. 공연별 안내 확인.

**Q10. 전화·현장 예매도 가능?**
A. 네. 국립극장 02-2280-4114, 국립국악원 02-580-3300, 예술의전당 1668-1352, 세종문화회관 02-399-1000, 정동극장 02-751-1500. '장애인 우대' 알리면 됨.

**Q11. 매진되면 우대 좌석도?**
A. 우대 좌석도 전체 좌석 중 일부. 매진 시 먼저 마감 가능. 예매 첫날 빠르게 접속 권장.
