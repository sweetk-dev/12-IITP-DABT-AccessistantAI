# B006. 이동통신 요금 감면 (기본료·통화료·데이터 35%, 가입비 면제)

> 전단지 원문: **"이동통신요금 할인(신규 가입비 면제, 기본요금 및 국내통화료 35%)"**
> 구분: 심한장애(중증) / 심하지 않은 장애(경증) — 전국공통, ⑥ 번 항목
> 최종 확인: 2026-05-19 (v1.0.0)

---

## 1. 한눈에 보기

| 항목 | 내용 |
|---|---|
| **무엇을** | 이동전화 **월정액(기본료) 35% + 국내 음성통화료 35% + 데이터 통화료 35%** 감면 + **신규 가입비 100% 면제** |
| **누구에게** | 「장애인복지법」에 따라 등록된 장애인 **본인 명의** 1회선. 장애인복지시설·단체는 단체당 2회선 |
| **어떻게 받는지** | 통신사 가입(본인 명의) → 복지할인 신청 → 자격 자동검증 → 다음 청구분부터 자동 감면. 신규 가입 시 동시 신청 가능 |
| **어디서 신청** | 이통3사 공동 **ARS 1523**(무료, 가장 권장) / 정부24·복지로 일괄신청 / 통신사 홈페이지·앱 / 직영점·대리점 / 동주민센터 |
| **비용** | 신청 무료. 신규 가입자는 **가입비 면제** |
| **유효기간** | 장애 자격과 연동. 영구 판정자는 만료 없음. **번호이동·명의변경 시 재신청 필수** |
| **감면 상한** | SKT 월 최대 28,600원(VAT 포함) / LGU+ 사례별 약 11,000원 안내 / KT·알뜰폰 별도 상한 표기 없음 |

## 3. 대상자 자격

- **등록 장애인 본인 명의 1회선**만 적용(다른 통신사 회선 포함 1개).
- **중증/경증 구분 없음** — 장애 정도와 무관하게 동일.
- 장애인복지시설·복지단체: 단체당 **2회선**(통합 청구 시).
- **유선·인터넷과 달리 '세대' 기준이 아닌 '개인' 기준** — 가족 명의 회선은 그 가족이 장애인 본인일 때만 가능.
- **외국인 등록 장애인**: 「장애인복지법」 제32조의2 등록자 동일 적용.

## 4. 받는 방식 (How to use)

### 감면 내용 요약

| 항목 | 감면율 | 비고 |
|---|---|---|
| **월정액(기본료)** | 35% | 모든 요금제 |
| **국내 음성통화료** | 35% | 영상통화·국제전화 제외 |
| **데이터 통화료** | 35% | 기본제공량 초과분 |
| **신규 가입비** | **100% 면제** | 신규 개통 시 |

### 신청 → 적용 흐름

1. **ARS 1523** (가장 빠름, 이통3사 공동, 무료) → 본인인증 → 자격 자동확인 → 신청 완료.
2. **정부24·복지로** 온라인 일괄신청 → 공동인증서·간편인증 로그인 → '요금감면 통합신청' → 통신사·서비스 선택 → 행정정보 공동이용 동의 → 통신사 시스템 반영(1~3영업일).
3. **통신사 홈페이지·앱**: T world / KT닷컴 / LG U+ 홈페이지에서 본인인증 후 복지할인 신청.
4. **방문 신청**: 직영점·대리점에 신분증·복지카드 지참, 신청서 작성.
5. **알뜰폰**: 각 사 고객센터·홈페이지·앱 '복지할인'·'요금감면' 메뉴.
6. **적용**: KT는 신청 당월 청구분부터, SKT·LGU+·알뜰폰은 보통 다음달부터.

### 감면 대상이 아닌 것
- **단말기 할부금**(별도 단말 약정에 따른 별도 청구)
- **영상통화·국제전화·정보료(060/080)**
- **부가서비스**(컬러링, 발신번호표시, 무선인터넷 콘텐츠 등)
- **소액결제·콘텐츠 요금**(앱마켓·결제대행)
- 선불 요금제(Prepaid) — 대부분 미적용

## 5. 통신사별 적용

- 감면: 월정액·국내음성통화료·데이터통화료 각 35%, 가입비 면제
- **월 최대 감면액 28,600원(VAT 포함)**
- 회선 제한: 개인 1회선/단체 2회선
- 복지할인 중복 적용 불가
- 신청: T world 홈페이지·앱 / 고객센터 1599-0011(유료), 휴대폰 114(무료) / ARS 1523
- 청각·언어 장애인용 '장애인전용 요금제' 별도 운영

### KT
- 감면: 동일하게 각 35%, 가입비 면제
- 결합할인-복지할인 우선순위:
  - **장애인·유공자·기초연금수급자**: 결합할인 우선 → 복지할인 추가
  - **기초생활수급자**: 복지할인 우선 → 결합할인

- 감면: 기본료·통화료 35% (공식 안내)
- 사례별 월 상한 약 11,000원 안내 사례 있음
- 회선 제한: 개인 1회선/단체 2회선
- 신청: 복지할인 전용 ARS **1523**(무료) / 고객센터 1544-0001 / 홈페이지 / 직영점·대리점

- 신청: 각 사 고객센터·홈페이지·전용 앱의 '요금감면'·'복지할인' 메뉴

## 7. 유효기간·갱신

- 장애 자격이 영구 판정인 경우 별도 만료 없음.
- 재판정 대상자: 재판정 기한 만료 시 통신사가 자격 재확인 안내(문자·청구서). 재판정 후 자격 유지 확인 시 자동 갱신.
- **명의 변경·번호이동·통신사 변경 시 재신청 필수** — 자동 이관 안 됨.
- 통신사 약관에 따라 1~3년 주기로 자격 재검증 안내.

## 8. 자주 묻는 질문 (FAQ)

**Q1. 정확히 얼마나 할인되나요? 35%가 세 군데 다 따로 적용?**
A. **각각 별도** 적용입니다 — 월정액 35% + 국내음성통화료 35% + 데이터통화료 35%. 신규 가입 시 가입비도 면제. 다만 월 상한이 있어 SKT는 월 최대 **28,600원(VAT 포함)**, LGU+는 사례별 약 11,000원 안내. KT·알뜰폰은 별도 상한 표기 없이 35% 적용.

**Q2. 가족(부모·배우자) 명의 회선도 감면되나요?**
A. **이동통신은 본인 명의 1회선에만 적용**됩니다(유선·인터넷의 '세대' 기준과 다름). 가족 명의 회선은 그 가족이 장애인 본인이어야만 가능. 단, 미성년 장애인은 보호자 대리 가입·신청 가능.

**Q3. 회선 2개 모두 감면되나요?**
A. **개인은 1회선만** 감면 가능(다른 통신사 회선 포함 통합 1개). 단체(시설·복지단체)는 2회선까지.

**Q4. 어디서 가장 빨리 신청하나요?**
A. **이통3사 공동 ARS 1523**(무료)이 가장 빠르고 간편. 본인인증만 거치면 자격 자동확인·신청 가능. 온라인은 정부24·복지로·통신사 홈페이지에서.

**Q5. 알뜰폰(MVNO)도 35% 받을 수 있나요?**
A. **네.** 모든 알뜰폰 사업자는 「전기통신사업법」 제4조에 따라 복지할인 제공 의무. 적용 범위·신청 절차는 사업자별 차이가 크며, 청각장애인 문자이용료 70% 할인 등 별도 혜택을 운영하는 사업자도 있음 → 가입 전 사업자 고객센터에 자격·할인액·신청 방법 사전 확인.

**Q6. 선택약정 25%와 복지할인 35%를 같이 받을 수 있나요?**
A. **일반적으로 중복 적용 가능**. 선택약정은 단말 지원금 대신 받는 요금제 할인, 복지할인은 장애 자격에 따른 별도 감면이므로 함께 적용. 정확한 계산은 통신사 고객센터 시뮬레이션 요청.

**Q7. 결합상품(인터넷+TV+모바일)도 결합할인+복지할인 둘 다 받을 수 있나요?**
A. 통신사별로 다름.
- **KT**: 장애인·유공자·기초연금 → 결합 우선 후 복지 추가 / 기초생활수급자 → 복지 우선 후 결합
- **SKB·LGU+**: 약관에 따라 중복 불가 또는 우선 적용 차이 → 가입 시 시뮬레이션 요청

**Q8. 단말기 할부금이나 콘텐츠 요금도 감면되나요?**
A. **아닙니다.** 단말기 할부금·소액결제·콘텐츠·영상통화·국제전화·정보료·부가서비스는 감면 대상 아님. 월정액·국내음성·데이터만.

**Q9. 청각·언어 장애인 전용 요금제가 있나요?**
A. 있습니다. SKT는 영상통화·문자 중심 '**장애인전용 요금제**'를 운영하고 이 요금제 사용 시에도 35% 복지할인 추가 적용. 일부 알뜰폰은 청각장애인 **문자이용료 70% 할인** 별도 제공.

**Q10. 신청 후 언제부터 할인되나요?**
A. **KT는 신청 당월** 청구분부터. SKT·LGU+·알뜰폰은 신청월 다음달부터가 일반적. 정부24 일괄신청은 통신사 반영 1~3영업일.

**Q11. 번호이동(통신사 변경)하면 감면이 그대로 이관되나요?**
A. **자동 이관 안 됨** → 새 통신사에서 재신청 필수. 정부24 일괄신청을 다시 이용하면 행정정보 공동이용으로 빠르게 처리.

**Q12. 신청 안 한 기간도 소급 환급되나요?**
A. **원칙적으로 소급 적용 불가**. 일부 통신사는 '신청월 또는 3개월 이내' 소급 정책 운영 → 가입 통신사 문의.

**Q13. 외국인 등록 장애인도 같은 혜택?**
A. 「장애인복지법」 제32조의2 등록자 동일 적용. 외국인등록증·복지카드 사본 필요.

## 9. 부정사용 시 제재

- 장애인 자격이 아닌 자가 부정 감면을 받은 경우 **차액 환수(추징)** 및 통신사 약관에 따른 서비스 이용 제한.
