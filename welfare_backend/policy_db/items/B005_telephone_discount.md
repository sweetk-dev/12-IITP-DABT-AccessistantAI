# B005. 유선전화(시내·시외·114) 요금 감면

> 전단지 원문: **"전화요금 50% 할인(시외통화 3만원이내, 114요금 전액 면제)"**
> 구분: 심한장애(중증) / 심하지 않은 장애(경증) — 전국공통, ⑤ 번 항목
> 최종 확인: 2026-05-19 (v1.0.0)

---

## 1. 한눈에 보기

| 항목 | 내용 |
|---|---|
| **무엇을** | 유선전화 시내·시외 **통화료 50% 감면**(시외는 월 3만원 한도) + **114 안내요금 100% 면제** |
| **누구에게** | 「장애인복지법」에 따라 등록된 장애인 본인 또는 **그 장애인이 속한 세대**(가족 명의 회선도 가능). 장애인복지시설·단체도 포함 |
| **어떻게 받는지** | 통신사에 신청 → 자격 자동검증 → 다음 청구분부터 자동 감면(KT는 신청 당월부터). **사후 환급 아님** |
| **어디서 신청** | 정부24·복지로 일괄신청 / 이통3사 공동 ARS **1523** / KT 100·SKB 106·LGU+ 1544-0001 / 대리점 방문 / 동주민센터 |
| **비용** | 신청 무료 |
| **유효기간** | 장애 자격과 연동(영구 판정자는 만료 없음). 명의·통신사·이사 변경 시 재신청 |

## 2. 근거 법령

| 법령 | 조문 | 비고 |
|---|---|---|
| 장애인복지법 | 제30조(경제적 부담의 경감) | 일반 근거 |
| 장애인복지법 시행령 | 제17조 및 별표2 | 공공시설 등의 이용요금 감면 기준 |
| 전기통신사업법 | 제4조 제3항 | 보편적 역무·사회적 약자 요금감면의 직접 근거 |
| 전기통신사업법 시행령 | 제2조 제1항 제3호·제2항 제3호 | 장애인 통신요금 감면 대상·내용 명시 |

> 「전기통신사업법」 시행령 별표상 시내전화·시외전화·114·인터넷전화·인터넷가입자접속 서비스는 **장애인이 속한 세대를 감면 대상**으로 한다고 규정.

---

## 3. 대상자 자격

- **등록 장애인** 본인 또는 **장애인이 속한 세대**(가족 명의 회선도 가능, 등본으로 세대 확인).
- **중증/경증 구분 없음** — 장애 정도와 무관하게 동일 감면.
- **장애인복지시설·복지단체** 명의 회선도 감면 대상(시설 등록증·사업자등록증 제출).
- **외국인 등록 장애인**: 「장애인복지법」 제32조의2에 따른 등록자 동일 적용.
- 회선 수 제한은 통신사 약관에 따름(보통 1세대 1회선).

## 4. 받는 방식 (How to use)

### 감면 내용 요약

| 서비스 | 감면율 | 한도 |
|---|---|---|
| **시내전화 통화료** | 50% | 한도 없음 |
| **시외전화 통화료** | 50% | **월 3만원 통화료까지** (최대 감면액 15,000원) |
| **114 안내요금** | **100% 면제** | 횟수 제한 없음 (안내 후 자동 연결되는 통화료는 별도 부과) |
| **인터넷전화(070) 통화료** | 50% (시내·시외 통합) | 통신사 약관에 따름 |

### 신청 → 적용 흐름

1. **정부24·복지로 일괄신청** (가장 권장)
   - 공동인증서·간편인증으로 로그인 → '요금감면 통합신청' → 통신사·서비스 선택 → 행정정보 공동이용 동의 → 자동 자격조회·접수.
   - 통신사 시스템 반영 1~3영업일.
2. **전화 신청**
   - 이통3사 공동 **ARS 1523** (장애인·기초생활수급자 자격 자동확인·신청)
   - KT: 유선 100 / 휴대폰 114 / SKB: 106 / LGU+: 1544-0001
3. **방문 신청**: 통신사 대리점에 신분증·복지카드 지참, 신청서 작성.
4. **자격 자동검증**: 행정안전부 주민서비스 통합정보시스템에서 자격 확인.
5. **적용**: KT는 당월 청구분부터, SKB·LGU+는 통상 다음달부터.

### 감면 대상이 아닌 것
- 국제전화(001/002 등)
- 060/080 정보료
- 부가서비스(컬러링·발신번호표시 등)
- **기본료(월정액)** — 본 항목은 '통화료'만 감면

## 7. 유효기간·갱신

- 장애 자격이 영구 판정인 경우 별도 만료 없음 → 통신사가 자동 유지.
- **재판정 대상자**: 재판정 기한 만료 시 통신사가 청구서·문자로 자격 재확인 안내. 재판정 후 자격 유지가 확인되면 자동 갱신(통신사에 따라 재신청 요구).
- **명의 변경·이사·통신사 변경 시 재신청 필수** — 자동 이관되지 않음.
- 통신사 약관에 따라 1~3년 주기로 자격 재검증 안내가 올 수 있음.

## 8. 자주 묻는 질문 (FAQ)

**Q1. 기본료(월정액)도 할인되나요?**
A. 유선전화는 **기본료는 감면 대상이 아닙니다**. 시내전화 통화료 50%, 시외전화 통화료 50%(월 3만원 한도), 114 100% 면제만 적용. 이동통신은 별도 항목(⑥번)으로 기본료·통화료·데이터 35% 감면이 별도 적용됩니다.

**Q2. 가족 명의 회선도 신청 가능한가요?**
A. 가능합니다. 「전기통신사업법 시행령」은 유선전화·인터넷전화·인터넷의 경우 '**장애인이 속한 세대**'를 감면 대상으로 규정합니다. 같은 세대원이면 가능하며, 등본으로 세대 확인(정부24 일괄신청 시 자동 확인).

**Q3. 한 집 회선 2개 모두 감면되나요?**
A. 원칙적으로 **세대당 1회선**입니다. 두 회선 모두 받으려면 통신사 사전 문의 필요.

**Q4. 어디서 가장 빨리 신청하나요?**
A. **정부24·복지로** 온라인 일괄신청(공동인증서·간편인증 즉시) 또는 **ARS 1523**(이통3사 공동, 무료).

**Q5. 결합상품 쓰는데 결합할인+복지할인 둘 다 받을 수 있나요?**
A. 통신사별로 다릅니다. **SKB는 중복 불가**(큰 1가지만), **KT는 장애인은 결합할인 우선 적용 후 복지할인 추가**, **LGU+**는 약관에 따라 다름 → 가입 시 고객센터에 상세 시뮬레이션 요청.

**Q6. 국제전화·060/080도 할인되나요?**
A. **아닙니다.** 국제전화·정보료·부가서비스는 본 항목 감면 대상이 아닙니다.

**Q7. 신청 후 언제부터 할인되나요?**
A. **KT는 신청 당월** 청구분부터, SKB·LGU+는 통상 다음달부터. 정부24 일괄신청은 통신사 반영 1~3영업일.

**Q8. 이사 가면 감면이 자동 이관되나요?**
A. **자동 이관 안 됨** → 번호 변경·이사·통신사 변경 시 재신청 필요.

**Q9. 신청 안 한 기간도 소급 환급되나요?**
A. **원칙적으로 소급 적용 불가**. 일부 통신사는 '신청월 또는 3개월 이내' 소급 정책 운영 → 가입 통신사 문의.

**Q10. 장애인복지시설·단체 명의 전화도 감면되나요?**
A. **네.** 시설 등록증·단체 등록증·사업자등록증 첨부 후 영업창구 신청.

**Q11. 114 안내요금 면제는 횟수 제한 있나요?**
A. **없습니다.** 횟수 제한 없이 안내요금(약 100원/회) 100% 면제. 단, **안내 후 자동 연결 통화**의 통화료는 별도 부과.

**Q12. 외국인 등록 장애인도 같은 혜택?**
A. 「장애인복지법」 제32조의2에 따른 등록자 동일 적용. 외국인등록증·복지카드 사본 필요.

## 9. 부정사용 시 제재

- 장애인 자격이 아닌 자가 부정으로 감면을 받은 경우 **차액 환수(추징)** 및 통신사 약관에 따른 서비스 이용 제한.
- 「전기통신사업법」 사회적 약자 요금감면 부정수급 적발 시 감면 자격 **1~3년 제한** 가능.
- 행정안전부 주민서비스 통합정보시스템의 자격 자동 검증으로 자격 상실 후 부정 수령 시 자동 환수 청구.
